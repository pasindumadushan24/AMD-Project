QuickMarket Mobile

QuickMarket Mobile is a mobile marketplace application developed using React Native Expo and Firebase for the ITS 2127 - Advanced Mobile Developer (AMD) final project.

Features

* User Registration and Login
* Create Advertisements
* View Advertisements
* Delete Advertisements
* Upload Product Images
* Category Filtering
* User Profile Management
* Firebase Authentication
* Firebase Firestore Database
* Firebase Storage Database

Technologies Used

* React Native
* Expo
* TypeScript
* Firebase Authentication
* Firebase Firestore
* Firebase Storage
* React Navigation
* React Context API

Installation

Clone the repository:

bash
git clone https://github.com/YOUR_USERNAME/QuickMarketMobile.git


Install dependencies:

bash
npm install


Run the project:

bash
npx expo start


Firebase Setup

Create a Firebase project and enable:

* Authentication
* Firestore Database
* Storage

Then add your Firebase configuration to:

bash
src/firebase/config.ts


APK Build

Generate an Android APK using:


bash
eas build -p android --profile preview
